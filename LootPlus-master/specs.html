Specs

Purpose

A Fun Game.

Our goal is to make a fun, interactive social gaming experience in a browser. Simple as that.

Use Case

Landing/Login Page

  • When the user reaches the landing page, they are greeted with patch notes and news about the latest version of the game.
  • The user has the option to login or continue as guest.
  • If the user chooses to sign in, a drop down appears where they can input credentials.
  • Once authentication is completed, the user is taken directly to the Game Page.
  • If the user chooses to continue as guest, the user is taken directly to the Game Page.

Game Page

  • The Game Page allows the user to play Loot Plus.
  • The user will have access to:
    • “Start New” – Starts a new save file and overrides any previous saves.
    • “Load Game” – Reloads previous save if user is signed in.
  • Once the user starts a new game or loads a previous save, they can access and play the game.

Forum Page

  • Users can access the forum page to view posts and discussions by other players.
  • The user must be signed in to post or make comments.

Credits/Information Page

  • This page will contain bios of the team and credits we owe to any other help we gathered along the way.

Deliverables

Page = Deliverable

Our deliverables are simply each of the pages of the site.

  • A home page with news and patch notes for the game.
  • A forum for questions and chatting.
  • The game.
  • Honorable mention: a credits page for the people that developed the site.

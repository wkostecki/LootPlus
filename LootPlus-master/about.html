Summary of Project

Game

A working prototype of the game can be found at the bottom of this page.

  • Infinite progression clicker-style game.
  • Mock the practice of lootboxes in games from recent years.
  • Variable interval reward schedule
  • Includes collectibles and rarity tiers of currencies to entice play.
    • Junk (every box)
    • Common (every other box)
    • Uncommon (every 10 boxes)
    • Rare (every 100 boxes)
    • Super Rare (every 1,000 boxes)
    • Ultra Rare (every 10,000 boxes)
    • Uber Rare (every 100,000 boxes)
  • Upgrade lootbox income with purchases using the various currencies.
  • Quick access on any device.
  • Progress is saved to a user login.

Forum

  • Central location for users to ask questions if they are lost.
  • Develop a community of players.
  • Allow developers to interact directly with players.
  • Users must login to post using the same info that saves game progression.

Home

  • News page for the game.
  • Updates and patch notes for changes.
